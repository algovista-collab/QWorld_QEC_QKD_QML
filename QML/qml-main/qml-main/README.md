## QWorld's Quantum Machine Learning (QML) module

The Quantum Machine Learning (QML) module is a tutorial that combines core QML concepts with practical notebook-based exercises.

This module uses **Qiskit Machine Learning** and **PennyLane**.

## Prerequisite
Students should have completed [QBronze](https://qworld.net/workshop-bronze/) material or a similar course.

Basic knowledge of linear algebra, Python, and introductory machine learning is recommended.

## Software Requirements
Jupyter Notebook (or JupyterLab), Python 3, Qiskit, Qiskit Machine Learning, and PennyLane.

## Content
Please refer to the [Contents](contents.ipynb) notebook.

## Quick Setup
To ensure you have the required dependencies for running this QML tutorial, install from `requirements.txt`:

```bash
pip install -r requirements.txt
```

You can run this command in your preferred environment (Conda, venv, or system Python).
For full installation notes and version checks, see [0.1 Installation](chapters/00-introduction/01-installation.ipynb).
